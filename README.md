# Music Member Recommender

## 概要

バンドメンバー募集サービスを想定し、ユーザー属性と自己紹介文から相性の良いメンバーを推薦するシステムです。

## 背景

バンドメンバー募集では、楽器・ジャンル・地域・活動目的・活動頻度のミスマッチが起こりやすいです。
本システムでは、ユーザー情報を特徴量化し、類似度に基づいて候補者を提示します。

## 使用技術

Python / pandas / scikit-learn / FastAPI / Docker

## 推薦手法

内容ベース推薦
TF-IDF
Cosine Similarity

## 工夫点

推薦理由の表示
活動地域・目的・頻度を考慮
APIとして利用可能な構成
